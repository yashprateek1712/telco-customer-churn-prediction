# Telco Customer Churn Prediction

This project predicts customer churn using machine learning models on the Telco Customer Churn dataset.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Final Model
Logistic Regression was selected as the final model because it showed the best generalization performance with minimal overfitting.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Preprocessing using ColumnTransformer
- Model Training
- Hyperparameter Tuning
- Model Evaluation
- Model Serialization using Pickle
